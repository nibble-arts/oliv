<h1>Template Format</h1>

<p>Die Templates für das Seitenlayout sind als XML-Datei abgelegt. Die Zeichenkodierung muss dabei UTF-8 sein. Das File muss folgenden Richtlinien entsprechen:</p>

<pre>
<site>
  <area name='area_name'>
  * <option id='parameter'>value</option>     # optionale weitere Parameter
  *             ...                           # optionale weitere Parameter  
  </area>
</site>
</pre>

<p>Die area Bereiche können verschachtelt sein, wie es für den Aufbau der div-Tags nötig ist.</p>

<pre>
<site>
  <area>                        # main area
    <option id =''></option>

    <area>                      # sub area
      <option id =''></option>
    </area>
  </area>
</site>
</pre>
