Template Format

Die Templates für das Seitenlayout sind als XML-Datei abgelegt. Die Zeichenkodierung muss dabei UTF-8 sein. Das File muss folgenden Richtlinien entsprechen:


  
     # Name des div bereichs
     # Styleangaben
  *  # optionale weitere Parameter
  *             .  # optionale weitere Parameter

Die area Bereiche können verschachtelt sein, wie es für den Aufbau der div-Tags nötig ist.


  # main area
  
    

    # sub area
